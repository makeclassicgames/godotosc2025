# Taller de Godot

En la siguiente página, puedes encontrar un taller sobre el motor godot impartido inicialmente en la OpenSouthCode 2025 y ampliado para impartirse en el C.E. Desarrollo de Videojuegos y Realidad Virtual del IES Al-Andalus.

En las siguientes páginas, podrás encontrar información sobre como crear un sencillo videojuego utilizando el motor de videojuegos libre Godot.

Utiliza el menú de la izquierda para ver cada uno de los apartados de este taller.
