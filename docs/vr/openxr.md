# OpenXR

OpenXR es una API de código abierto desarrollada por el grupo Khronos que proporciona una interfaz unificada para el desarrollo de aplicaciones de realidad virtual (VR) y realidad aumentada (AR). OpenXR permite a los desarrolladores crear aplicaciones que pueden ejecutarse en una amplia variedad de dispositivos VR y AR sin necesidad de realizar cambios significativos en el código.

En Godot, el soporte para OpenXR se ha integrado para facilitar el desarrollo de experiencias de VR y AR. Esto significa que puedes crear aplicaciones de realidad virtual utilizando Godot y aprovechar las capacidades de OpenXR para garantizar la compatibilidad con una amplia gama de dispositivos.
