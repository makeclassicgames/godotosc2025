# Introducción a 3D con Godot

En esta sección, aprenderás los conceptos básicos de cómo trabajar con gráficos 3D en Godot. Cubriremos temas como la creación de escenas 3D, la manipulación de objetos, la iluminación y el uso de materiales.

## Creación de una escena 3D

Para crear una escena 3D en Godot, sigue estos pasos:

1. Abre Godot y crea un nuevo proyecto.
2. En el panel de la izquierda, haz clic derecho en el nodo raíz y selecciona "Add Child Node".
3. Busca y selecciona ```Node3D``` para crear un nodo 3D.
4. Renombra el nodo a "MainScene" o cualquier nombre que prefieras.

Es importante mencionar que en Godot, las escenas pueden contener otros nodos, lo que te permite organizar tu proyecto de manera eficiente. Puedes agregar nodos como ```MeshInstance3D``` para objetos 3D, ```Camera3D``` para la cámara y ```Light3D``` para la iluminación.

En esta sección, nos centraremos en cómo agregar y manipular objetos 3D, así como en la configuración de la iluminación y los materiales para crear una escena atractiva.

## Manipulación de objetos 3D

Una vez que hayas creado tu escena 3D, puedes agregar objetos como ```MeshInstance3D``` para representar modelos 3D. Puedes importar modelos desde software de modelado 3D o usar los modelos predeterminados de Godot. Para manipular estos objetos, puedes usar las herramientas de transformación (mover, rotar y escalar) disponibles en el editor de Godot.

Durante esta sección, aprenderás a manipular objetos 3D para crear escenas dinámicas y atractivas. También veremos como configurar las propiedades y añadir nodos para la física y colisiones, lo que es esencial para la interacción en juegos 3D.
